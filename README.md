# Python-Hangman-Game
Simple Hangman game built in Python while revisiting programming fundamentals.

This project is a simple Hangman game implemented in Python.

The goal was to revisit Python fundamentals including:
- Functions
- Loops
- Conditional logic
- Handling user input
- Managing program state

Features:
- Random word selection
- Board display
- Duplicate guess detection
- Win / lose logic
- Play again functionality

This project was built in a Jupyter Notebook.
